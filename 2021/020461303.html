主要用来ctf解题
个人概括起RSA来说就是：每个人都有公钥e，可以根据公钥e加密信息，但只有持有私钥d的我才能对信息进行解密。

1互质关系

两个正整数除1以外没有公因子 （所以不仅仅是两个质数可以构成互质关系）

2 python中gmpy2库的使用

①gmpy2.mpz(n)………………..初始化一个大整数
②gmpy2.mpfr(x)………………..初始化一个高精度浮点数x
③gmpy2.iroot(x,n)……………..x开n次根
④c = gmpy2.powmod(m,e,n).幂取模，结果是 c = (m^e) mod n
⑤gmpy2.is_prime(n)………….素性检测
⑥gmpy2.gcd(a,b)………………欧几里得算法，最大公约数
⑦gmpy2.gcdext(a,b)………….扩展欧几里得算法
⑧gmpy2.next_prime(p)…………顾名思义
⑨d = gmpy2.invert(e,n)………求大整数e模n的逆元d，invert（e，n）返回d使得e * d == 1 mod n

3 基础解题

①一般给出两个素数p,q(若给出的是n，可以去在线分解大整数 )

②计算公共模数 n
n = p * q

③欧拉函数φ(n)
φ(n) = （p-1）*（q-1）

④公钥e
1 < e < φ(n)

⑤私钥d
(d * e) mod φ(n) = 1
d * e ≡ 1 mod ((p-1) * (q-1))
≡的意思:a ≡ b mod c 等价于 b = a mod c
可用RSA-Tool2 by tE工具根据p,q,e解出(使用工具时e要进行10进制到16进制的转化)

⑥加密：密文c
c = (m^e) mod n

⑦解密：明文m
m = (c^d) mod n

若题目说了格式为flag{}则直接上交
若未说，则转换成字符串的脚本如下： 

import gmpy2

4进阶解题

①私钥d
d = gmpy2.invert(e,phi)

②（未）给出dp，dq
dp ≡ d mod (p-1)
dq ≡ d mod (q-1)

③计算mp,mq,I 

mp = (c^dp) mod p
mq = (c^dq) mod q  
I = gmpy2.invert(q,p)

④ 明文m
m = (((mp-mq)*I)%p)q + mq

5 已知e,n1,n2,c1,c2

①求出n1和n2的最大公约数p
p = gmpy2.gcd(n1,n2)
②计算q
q = n1//p (不必考虑q1,q2的情况，最后解出的字符串都一样)
③计算phi,d,m 

phi = （p-1）*（q-1)
d = gmpy2.invert(e,phi)    
m = gmpy2.powmod(c1,d,n1)

7 低加密指数e攻击（e相对n来说很小）

常用于e = 3 的情况
①m^3 < n
c = m^3
②m^3 > n
c = (m^3) - i * n
即找到i,使得c+i * n可开三次方

10 低加密指数广播攻击

当e比较小，且使用了相同的e给若干个接收者发送相同的信息，有以下等式成立： 

c1 = pow(m, e, n1)  
c2 = pow(m, e, n2)  
c3 = pow(m, e, n3)

对上述等式运用中国剩余定理，在e=3时，可以得到：
c _ x = pow(m, 3, n1 * n2 * n3)
通过对 c_x进行三次开方可以得到明文

13 e(为偶数)与phi不互素
